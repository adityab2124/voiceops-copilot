# VoiceOps Copilot

Streamlit prototype for a Technical Account Manager workflow across voicebot account health, QA triage, prompt fixes, engineering handoff, compliance responses, and calling-window audits.

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

Optional Gemini features use `GEMINI_API_KEY` in Streamlit secrets. The app includes offline fallbacks for demo flows.
